# 🧠 Personality Test

A modern **Personality Test** web application built with **Python** and **Flask** that evaluates a user's personality traits based on their responses to a series of questions. Using predefined scoring logic and psychological assessment techniques, the application analyzes user behavior and generates a detailed personality profile along with personalized insights.

> **Discover your personality traits through intelligent psychological assessment.**

# ⚙️ Installation

## 1. Clone the Repository

```bash
git clone https://github.com/subham-paul/Personality_Test.git
```

```bash
cd Personality_Test
```

---

## 2. Create a Virtual Environment

### Windows

```bash
python -m venv venv
```

Activate

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv venv
```

Activate

```bash
source venv/bin/activate
```

---

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4. Run the Application

```bash
python app.py
```

or

```bash
flask run
```

# ⚙️ How It Works

### Step 1 — Start the Test

Users begin the personality assessment by answering a series of carefully designed questions.

---

### Step 2 — Submit Responses

Each answer contributes to a predefined scoring system representing different personality traits.

---

### Step 3 — Personality Analysis

The application evaluates the collected responses and calculates scores for various personality dimensions.

---

### Step 4 — Result Generation

Based on the highest-scoring traits, the system determines the user's personality profile and generates personalized insights.

---

### Step 5 — Display Report

The final report may include:

- 🧠 Personality Type
- 📊 Trait Scores
- 💡 Strengths
- 🚀 Areas for Improvement
- 🎯 Career Suggestions *(Optional)*
- 🤝 Communication Style *(Optional)*
